Taller Rápido de Datos Estructurados para SEO

Durante este taller revisamos el concepto de "fragmentos enriquecidos mediante datos estructurados", damos varios ejemplos prácticos de implementación y platicamos cómo este tipo de meta-datos se están convirtiendo en un estándar.

Para ver el taller completo Datos Estructurados para SEO visita Webtraining.Zone.

Ejecución de los ejemplos

Instalar Live Server para ver los demos de mejor manera

sudo npm install live-server -g

Clonar y ejecutar los ejemplos

git clone https://github.com/webtraining-zone/structured-data-seo.git
cd structured-data-seo
live-server .
